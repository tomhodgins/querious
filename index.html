Element Queries Demo

This is not a serious element query plugin, just an experiment to demonstrate what the concept of element queries is about, prove how support can be written in JavaScript, and to highlight the features and limitations of such an approach.

Limitations:

  • can only use PX units
  • can only apply one instance of each condition per element
  • doesn't support IE8, and likely some other browsers too
